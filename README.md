# CodeGenerator
Unique olarak belirli bir karakter kümesi içerisinden 8 haneli kod üreten. Sonrasında bu karakter kümesinin doğruluğunu check eden bir uygulamadır. Chek aşamasında üretilen kodun karakter kümesine bağlı olması ve 8 haneden az veya çok olmamasını kontrol eder.
Teknik döküman proje içerisine eklenmiştir.
